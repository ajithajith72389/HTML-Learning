Forms

Attributes Description
action used to specify a URL that processes the form submission.
method used to define which HTTP method to use when submitting the form.
target used to specify where to open the linked document.
autocomplete set whether the autocomplete for the form should be on or off.
enctype used to specify how the form input data should be encoded before sending it to the server.
novalidate while submitting the form, the form data should not be validated in an HTML document.
