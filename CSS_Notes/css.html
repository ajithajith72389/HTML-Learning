CSS Selectors

Basic Selectors

  1. Universal Selector (*)
  2. Element Selector (like <p>, <h1>, etc)
  3. Class Selector (.)
  4. ID Selector (#)

Combinator Selectors

  1. Descendant Selectors ( div p )
  2. Child Selector ( div > p )
  3. Adjacent Sibling Selector ( div + p )
  4. General Sibling Selector ( div ~ p )

Attribute Selector

  1. Presence Selector
  2. Attribute Value Selector
  3. Substring Matching

Pseudo classes

  1. : hover
  2. : focus
  3. : first -child
